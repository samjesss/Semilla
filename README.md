# Semilla

Herramienta educativa para personas que reciben remesas y quieren usar mejor el dinero que les sobra después de cubrir gastos básicos.

La app ayuda a:

- hacer un diagnóstico financiero corto;
- recibir una primera acción recomendada;
- calcular presupuesto, fondo de emergencia, deuda e interés compuesto;
- aprender conceptos financieros con lecciones sencillas.

## Comandos

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Enfoque de producto

Semilla prioriza una experiencia simple: primero sugiere un siguiente paso claro y luego deja calculadoras y lecciones como apoyo. Los módulos incompletos permanecen marcados como próximos temas para no mostrar contenido vacío.
